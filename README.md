## Development

### Installing Dependencies and Start Server

From within the root directory:

For git init:
```sh
git remote remove origin
git remote add origin https://github.com/...
```

For server:
```sh
npm install
npm run build
npm start
```

For dev:
```sh
npm run building
npm run starting
```
